Website Status Checker

Uses the Wayback Machine (archive.org) to check if a domain has been recently crawled and is still active.

How it works: The Wayback Machine crawls and archives billions of websites. If a domain was crawled recently it is almost certainly active. If it has not been crawled in over 2 years, or has never been archived, it is likely defunct, parked, or redirected. This runs directly in your browser with no installs needed.

Paste your account list

Tab-separated: Account Name + Website (copy from Excel), or just paste domains one per line.
Domains like google.com, acme.com, or full URLs all work.
